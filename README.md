# asdf-handy

mise/asdf plugin for [Handy](https://github.com/cjpais/Handy) â€” a local-first AI assistant.

## Install

```bash
mise plugins install handy https://github.com/TwistingTwists/handy-mise-plugin.git
mise install handy@latest
mise use -g handy@latest
```

## Usage

```bash
# install specific version
mise install handy@0.7.1
mise use -g handy@0.7.1

# upgrade
mise upgrade handy

# list available versions
mise ls-remote handy
```

## Supported platforms

- Linux x86_64 (amd64)
- Linux aarch64 (arm64)
